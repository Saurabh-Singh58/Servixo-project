# 🏠 Servixo - Home Services Booking Platform

## 📌 About the Project

Servixo is a web-based home service booking platform that connects users with trusted professionals like plumbers, electricians, tutors, carpenters, painters, and cleaners.

Users can browse services, view professionals, and book services easily through a clean and user-friendly interface.

This project includes user authentication (signup/login), session handling, and service category management.

---

## 🚀 Features

- 🔐 User Signup & Login system  
- 👤 Session-based authentication  
- 🧑‍🔧 Multiple service categories (Plumber, Electrician, Tutor, etc.)  
- ⭐ Ratings and reviews UI  
- 📍 Location display (demo data)  
- 🛒 Book Now functionality (frontend)  
- 📱 Responsive UI design

## ⚙️ How to Run Locally

1. Install XAMPP / WAMP  
2. Move project folder to `htdocs`  
3. Start Apache & MySQL in XAMPP  
4. Open phpMyAdmin and import `database.sql`  
5. Run project in browser:

http://localhost/servixo/
