# Servixo - Local Service Finder

Servixo is a modern web application that helps users find local service providers like plumbers, electricians, cleaners, and more.

## Features

- Browse services by category
- View popular services
- Search functionality
- Clean and responsive UI
- Service booking interface

##Tech Stack

- HTML
- CSS
- JavaScript

## Project Structure

- index.html -> Home page
- services.html -> All services
- about.html -> about page
- css/ -> Styles
- js/ -> Scripts
- images/ -> Assets

## Note

This project is created for learning and college purposes. All data used is dummy.
